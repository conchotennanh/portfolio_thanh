# 🌐 WordPress Portfolio - Thành

Đây là dự án xây dựng trang web Portfolio cá nhân sử dụng mã nguồn mở WordPress, được thực hiện cho bài tập môn Phát triển Web.

## 📌 Thông tin dự án
* **Họ tên:** Phạm Trung Thành
* **Dự án:** Portfolio Cá Nhân
* **Nền tảng:** WordPress
* **Hosting:** InfinityFree (hoặc Localhost)

## 🛠 Công nghệ sử dụng
* **Ngôn ngữ:** PHP
* **Cơ sở dữ liệu:** MySQL
* **Giao diện:** Tùy chỉnh trên nền tảng WordPress
* **Công cụ quản lý:** GitHub, XAMPP

## 📂 Cấu trúc thư mục
* `wp-content/`: Chứa các theme, plugin và hình ảnh của dự án.
* `wp-config.php`: File cấu hình kết nối Database.
* `.htaccess`: Cấu hình đường dẫn tĩnh (Pretty Permalinks).

## 🚀 Cách cài đặt
1. Clone repo này về máy.
2. Import file database `.sql` vào phpMyAdmin.
3. Cấu hình lại file `wp-config.php` cho khớp với database cục bộ.
4. Chạy dự án thông qua XAMPP hoặc các công cụ tương đương.
